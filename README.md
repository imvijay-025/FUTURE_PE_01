# FUTURE_PE_01 – Prompt Engineering Task 1

## Business Chosen
For this project, I selected a local business example called "Glow Beauty Salon" located in Chennai. The goal was to generate professional website copy for a beauty salon including homepage content, services page content, and call-to-action sections.

## Prompt Logic
Structured prompts were designed to generate specific sections of a website. Separate prompts were created for:
- Homepage copy (headline, subheadline, introduction)
- Services page content (service descriptions and benefits)
- Call-to-action sections (booking prompts and customer engagement)

Each prompt includes details such as business name, location, target audience, and tone. This ensures the AI generates relevant and high-quality website copy that can be used for real businesses.

## Tool Used
The AI tool used for generating the content in this project is ChatGPT.

## Project Structure
prompts – contains the prompts used for generating content  
outputs – contains the AI generated website copy
